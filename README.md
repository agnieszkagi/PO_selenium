# Automation Testing Of User Signup Form Using Selenium Webdriver

- Environment:
  ```
  Chrome wersja 88.0.4324.96,
  Ubuntu 20.04 LTS
  ```

- Requirements:
  ```
  selenium==3.141.0,
  ddt==1.4.1
  ```

- To run test, use command:
  ```
  make test
  ```
- To run black, use command:
  ```
  make lint
  ```
